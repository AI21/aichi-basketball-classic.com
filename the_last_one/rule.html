THE LAST ONE（ザ ラスト ワン）

  1. エントリー

THE LAST ONE 基本ルール

  1. チーム決め

    ・参加者全員で同時に3Point ｼｭｰﾄを放ち成功した者から順にチームを結成する
    ・参加者の人数が3の倍数にならない場合最後に残った者はそのゲームが終わるまでは参加できない。

  2. ゲーム(3on3方式)

    ・チーム数によってトーナメント及びリーグ戦を採用する
    ・11点先取下チームが勝利
    　※ 通常2pt→1pt.3pt→2pt
    ・全てのファウルルール及びアウトオブバウンズはトップからのスタート
    ・バスケットカウントの際は得点＋1pt＋ｵﾌｪﾝｽﾎﾞｰﾙ
    ・レフリー、スコアは試合に出場していないチームがすること

  3. 優勝者選出(リーグ戦)

    ・例えば4チームがトーナメントを行い1チーム(優勝権を獲得する3名)を選出、その3名が参加者を対象にドラフトし、新たに3チームを編成する。
    その3チームによるリーグ戦で勝率又は得失点差が高いチームの優勝権を持つプレーヤーが優勝【LAST ONE】となる

  4. 基本的にTHE LAST ONEでは試合の時間制限をとっていません。
